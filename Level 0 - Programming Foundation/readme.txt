Level 0: Programming Foundation

Before deep C++, understand these basic ideas:

1. What is programming?
2. What is a programming language?
3. Source code, compiler, linker, executable file
4. Difference between compiled and interpreted languages
5. What is C++?
6. C++ standards: C++98, C++03, C++11, C++14, C++17, C++20, C++23
7. Installing a compiler: GCC, Clang, or MSVC
8. Installing an editor or IDE: VS Code, Visual Studio, Code::Blocks, CLion
9. Writing your first program
10. Compiling and running a program
11. Understanding errors: syntax error, runtime error, logical error

How to study this level:

1. Read theory_concepts.txt first.
2. Follow practical_steps.txt on your computer.
3. Complete practice_tasks.txt one by one.
4. Write every command and every error in notes.txt.
5. Do not move to Level 1 until you can create, compile, run, and fix a simple C++ program yourself.

Files in this folder:

1. theory_concepts.txt
   Short theory for every foundation topic.

2. practical_steps.txt
   Step-by-step practical work for installation, first program, compiling, and running.

3. practice_tasks.txt
   Beginner tasks to build confidence.

4. notes.txt
   Your personal notes, errors, commands, and learning record.
