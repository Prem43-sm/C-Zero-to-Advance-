Level 1: C++ Basics

Goal:
Learn the basic parts of a C++ program and practice writing simple input/output programs.

Topics:

1. Basic structure of a C++ program
2. #include
3. Header files
4. main() function
5. Statements and semicolons
6. Comments: single-line and multi-line
7. cout for output
8. cin for input
9. endl and \n
10. Namespaces
11. using namespace std
12. Basic formatting of code

How to study this level:

1. Read theory_concepts.txt.
2. Open each .cpp file and understand the code.
3. Compile and run each program.
4. Change the code and observe the output.
5. Complete practice_tasks.txt.
6. Write mistakes and commands in notes.txt.

Files in this folder:

1. theory_concepts.txt
   Topic-by-topic explanation with code snippets.

2. practical_steps.txt
   Compile/run commands and practice flow.

3. practice_tasks.txt
   Tasks you should complete yourself.

4. notes.txt
   Your own learning notes.

5. example_01_basic_structure.cpp
6. example_02_output.cpp
7. example_03_input.cpp
8. example_04_intro_program.cpp
9. example_05_comments_formatting.cpp

Completion target:
You should be able to write a simple C++ program that prints output, takes input, uses comments, and follows basic formatting.
