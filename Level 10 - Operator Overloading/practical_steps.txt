Step 7: Modify and learn

For each suitable program:
1. Add another related operator.
2. Check whether the operator should modify the object.
3. Add const where appropriate.
4. Test chained operations.
5. Test equal, smaller, negative, and zero values.

Step 8: Error practice

Create and understand these errors:

1. Try to overload an operator using only built-in operands.
2. Forget const on an operator used with a const object.
3. Return a local object by reference.
4. Give postfix ++ no int parameter.
5. Return ostream by value from operator<<.
6. Access an invalid index without checking it.
