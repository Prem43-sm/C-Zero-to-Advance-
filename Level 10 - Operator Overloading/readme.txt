Level 10: Operator Overloading

Goal:
Make user-defined types work naturally with C++ operators while preserving
the operators' expected meaning.

Topics:

1. What is operator overloading?
2. Rules of operator overloading
3. Unary operator overloading
4. Binary operator overloading
5. Overloading using member functions
6. Overloading using friend functions
7. Stream insertion operator <<
8. Stream extraction operator >>
9. Assignment operator
10. Subscript operator []
11. Function call operator ()
12. Comparison operators

How to study this level:

1. Read theory_concepts.txt.
2. Compile and run every example program.
3. Identify each operator's operands and return type.
4. Complete practice_tasks.txt.
5. Write mistakes and observations in notes.txt.
