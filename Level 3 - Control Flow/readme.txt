Level 3: Control Flow

Goal:
Learn how to make decisions in C++ programs using conditions.

Topics:

1. if
2. if else
3. else if
4. Nested conditions
5. switch
6. Ternary operator
7. Truthy and falsy values
8. Input validation basics

How to study this level:

1. Read theory_concepts.txt.
2. Compile and run every example program.
3. Change the input values and predict the output.
4. Complete practice_tasks.txt.
5. Write your mistakes and observations in notes.txt.

Example programs:

1. example_01_if.cpp
2. example_02_if_else_even_odd.cpp
3. example_03_else_if_grade.cpp
4. example_04_nested_conditions.cpp
5. example_05_switch_day.cpp
6. example_06_ternary_operator.cpp
7. example_07_truthy_falsy.cpp
8. example_08_input_validation.cpp
9. example_09_largest_of_three.cpp
10. example_10_menu_calculator.cpp

Completion target:
You should be able to write programs that choose different actions based on user input.
