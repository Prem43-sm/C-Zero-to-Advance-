Level 4: Loops

Goal:
Learn how to repeat code using loops and solve repeated-action problems.

Topics:

1. Why loops are used
2. for loop
3. while loop
4. do while loop
5. Nested loops
6. break
7. continue
8. Infinite loops
9. Loop counters
10. Pattern printing

How to study this level:

1. Read theory_concepts.txt.
2. Compile and run every example program.
3. Change loop start, end, and step values.
4. Complete practice_tasks.txt.
5. Write mistakes and observations in notes.txt.
