Level 6 Practical Steps

Goal:
Practice arrays, matrices, and strings using real programs.

g++ example_01_array_basics.cpp -o example_01_array_basics
.\example_01_array_basics.exe

Step 8: Modify and learn

For each program:
1. Change the array size.
2. Change loop conditions.
3. Try negative numbers.
4. Try repeated values.
5. Try empty or one-word strings.
6. Write observations in notes.txt.

Step 9: Error practice

Create and fix these errors:

1. Access array index outside the valid range.
2. Use <= size instead of < size.
3. Forget #include <string>.
4. Mix cin and getline without handling the leftover newline.
5. Compare strings with wrong logic in palindrome program.
