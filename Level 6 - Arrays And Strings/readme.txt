Level 6: Arrays And Strings

Goal:
Learn how to store multiple values using arrays and how to work with text using strings.

Topics:

1. One-dimensional arrays
2. Array declaration
3. Array initialization
4. Accessing array elements
5. Updating array elements
6. Traversing arrays
7. Finding sum, average, minimum, maximum
8. Searching in arrays
9. Sorting basics
10. Two-dimensional arrays
11. Matrix input and output
12. Matrix addition
13. Matrix multiplication
14. C-style strings
15. std::string
16. String input problems
17. String functions
18. String traversal
19. Palindrome strings
20. Character functions

How to study this level:

1. Read theory_concepts.txt.
2. Compile and run every example program.
3. Change array sizes, values, and string inputs.
4. Complete practice_tasks.txt.
5. Write mistakes and observations in notes.txt.
