Step 8: Modify and learn

For each program:
1. Add one new data member.
2. Add one new member function.
3. Make data private.
4. Add validation in setter functions.
5. Write observations in notes.txt.

Step 9: Error practice

Create and understand these errors:

1. Try to access private data directly from main.
2. Forget semicolon after class closing brace.
3. Call a member function without an object.
4. Define static data member incorrectly.
5. Create a constructor with wrong class name.
