Step 7: Modify and learn

For each suitable program:
1. Add another derived class.
2. Add one virtual member function.
3. Call the function through a base reference.
4. Add override to derived implementations.
5. Predict constructor and destructor order.

Step 8: Error practice

Create and understand these errors:

1. Try to create an object of an abstract class.
2. Remove an override of a pure virtual function.
3. Misspell an overridden function while using override.
4. Access an ambiguous diamond-base member without virtual inheritance.
5. Pass a derived object by value and observe object slicing.
